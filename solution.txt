Challenge 1 — Qui utilise cette machine ?
Flag : PoC{PoC_W0rk$Hop,BrunoDeChezPoC}

Raisonnement :
J’ai utilisé le plugin envars pour lister les variables d’environnement de tous les processus, puis j’ai filtré sur « USERNAME »
Commande :
python vol.py -f vmss.core --profile=Win10x64_14393 envars | grep USERNAME
La sortie a révélé deux utilisateurs distincts : PoC_W0rk$Hop et BrunoDeChezPoC

Flag : PoC{7912+BrunoDeChezPoC}

Raisonnement :

Ligne de commande :
C:\Users\BrunoDeChezPoC\AppData\Local\Programs\Python\Python38\python.exe BadProgram
